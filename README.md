# Project setup frontend

**Prerequisites**
- Node.js v18+
- npm / yarn
- VideoSDK account
- VideoSDK JWT Token

**Clone**
- `git clone https://github.com/RizvanMalek/react-video-room-application`
- `cd react-video-room-application`

**Install dependencies**
- 'npm install' or 'yarn install'

**Copy ENVs**
- `cp .env.exampe .env`

**Run application**
- `npm run dev`

**How It Works**
- We can create or join a room using a inbuilt functionalites of the VideoSDK
